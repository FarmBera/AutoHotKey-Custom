# My Custom AutoHotKey Program

## Project Introduction

- 컴퓨터를 편하게 사용하기 위한 AutoHotKey 프로젝트
- 놀고 있는 Capslock 키를 다기능 Function 키로 활용하기 위하여 추진된 프로젝트

## Compiler Version

- v.1.37.01c1 U32 Ahk2Exe.exe
